# app_destroy_entity

Destroys an entity.

## Syntax

```cpp
void app_destroy_entity(app_t* app, entity_t entity);
```

## Function Parameters

Parameter Name | Description
--- | ---
app | The application.
entity | The type of entity to destroy.
