# rnd_seed

Initializes a seeded psuedo-random number generator.

## Syntax

```cpp
rnd_t rnd_seed(uint64_t seed);
```

## Function Parameters

Parameter Name | Description
--- | ---
seed | The seed to initialize `rnd_t` with.

## Return Value

Returns a ready-to-go psuedo-random number generator.
