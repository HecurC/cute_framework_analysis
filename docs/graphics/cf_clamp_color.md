Returns the component-wise clamp of `a` between `lo` and `hi`.

```cpp
CF_Color cf_clamp_color(CF_Color a, CF_Color lo, CF_Color hi)
```

Parameters | Description
--- | ---
a | The color.
lo | The min value for clamping.
hi | The max value for clamping.
