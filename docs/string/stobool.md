# stobool | [string](https://github.com/RandyGaul/cute_framework/blob/master/docs/string_readme.md) | [cute_string.h](https://github.com/RandyGaul/cute_framework/blob/master/include/cute_string.h)

Converts a string to a bool and returns it.

```cpp
#define stobool(s) cf_string_tobool(s)
```

## Remarks

Supports srings that start with "0x", "#", or no prefix.
