# sempty | [string](https://github.com/RandyGaul/cute_framework/blob/master/docs/string/README.md) | [cute_string.h](https://github.com/RandyGaul/cute_framework/blob/master/include/cute_string.h)

Returns whether or not the string is empty.

```cpp
#define sempty(s) cf_string_empty(s)
```

Parameters | Description
--- | ---
s | The string. Can be `NULL`.

## Remarks

Both "" and NULL count as empty.
