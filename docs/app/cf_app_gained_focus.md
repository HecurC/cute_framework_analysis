# cf_app_gained_focus | [app](https://github.com/RandyGaul/cute_framework/blob/master/docs/app/README.md) | [cute_app.h](https://github.com/RandyGaul/cute_framework/blob/master/include/cute_app.h)

Returns true if the app gained focus last frame.

```cpp
bool cf_app_gained_focus();
```

## Remarks

The app has focus if user inputs will go to the app, such as after clicking on or selecting the app.
