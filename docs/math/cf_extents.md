# cf_extents | [math](https://github.com/RandyGaul/cute_framework/blob/master/docs/math/README.md) | [cute_math.h](https://github.com/RandyGaul/cute_framework/blob/master/include/cute_math.h)

Returns the extents of an AABB (axis-aligned bounding box).

```cpp
CF_V2 cf_extents(CF_Aabb aabb)
```

## Remarks

Extents refer to width and height: `extents = { width, height }`.
