# cf_sound_set_is_paused | [audio](https://github.com/RandyGaul/cute_framework/blob/master/docs/audio_readme.md) | [cute_audio.h](https://github.com/RandyGaul/cute_framework/blob/master/include/cute_audio.h)

Sets the paused state for the sound.

```cpp
void cf_sound_set_is_paused(CF_Sound sound, bool true_for_paused);
```

Parameters | Description
--- | ---
sound | The sound.
true_for_paused | The pause state to set.

## Remarks

You can get a sound's paused state with [cf_sound_get_is_paused](https://github.com/RandyGaul/cute_framework/blob/master/docs/audio/cf_sound_get_is_paused.md).
