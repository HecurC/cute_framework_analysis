# cf_sound_get_volume | [audio](https://github.com/RandyGaul/cute_framework/blob/master/docs/audio/README.md) | [cute_audio.h](https://github.com/RandyGaul/cute_framework/blob/master/include/cute_audio.h)

Returns the volume of the sound.

```cpp
float cf_sound_get_volume(CF_Sound sound);
```

Parameters | Description
--- | ---
sound | The sound.

## Remarks

You can set a sound volume with [cf_sound_set_volume](https://github.com/RandyGaul/cute_framework/blob/master/docs/audio/cf_sound_set_volume.md), or upon creation with [cf_play_sound](https://github.com/RandyGaul/cute_framework/blob/master/docs/audio/cf_play_sound.md).
