# cf_sound_set_is_looped

Category: [audio](https://github.com/RandyGaul/cute_framework/blob/master/docs/api_reference?id=audio)  
GitHub: [cute_audio.h](https://github.com/RandyGaul/cute_framework/blob/master/include/cute_audio.h)  
---

Sets the looped state for the sound.

```cpp
void cf_sound_set_is_looped(CF_Sound sound, bool true_for_looped);
```

Parameters | Description
--- | ---
sound | The sound.
true_for_looped | The loop state to set.

## Remarks

You can get a sound's looped state with [cf_sound_get_is_looped](https://github.com/RandyGaul/cute_framework/blob/master/docs/audio/cf_sound_get_is_looped.md).
